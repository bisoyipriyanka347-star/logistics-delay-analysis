# 🚚 Logistics Delay Analysis

## 📌 Project Overview
This project analyzes logistics delivery data to identify key factors causing delivery delays and to understand how operational conditions impact delivery performance.

## 🎯 Problem Statement

The objective of this project is to answer:

- What percentage of orders are **delayed**?
- How do **weather conditions** affect delivery performance?
- Does **traffic congestion** increase delays?
- Is **waiting time at warehouses** contributing to delays?
- How does **inventory availability** impact deliveries?

## 🛠 Tools Used

- **SQL** – Data analysis
- **Power BI** – Dashboard creation
- **Excel / CSV Dataset** – Data source

## 📊 Key Insights

- **Bad weather conditions increased delivery delays significantly**
- **High traffic areas showed the highest delay percentage**
- **Higher waiting time correlated with delayed shipments**
- **Low inventory levels affected delivery processing**

## 💡 Business Recommendations

- Implement **traffic-aware route planning**
- Improve **warehouse dispatch efficiency**
- Monitor **inventory levels proactively**
- Use **weather-based delivery planning**
