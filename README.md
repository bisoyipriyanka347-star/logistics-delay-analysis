# logistics-delay-analysis
Analyzing delivery delays using SQL and Power BI

End-to-end Data Analysis Project using SQL and Power BI.

📌 Project Overview

This project analyzes logistics delivery data to identify the key factors causing delivery delays.
The goal is to understand how weather conditions, traffic status, waiting time, and inventory levels impact delivery performance and to derive insights that can help improve logistics efficiency.

🎯 Problem Statement

Delivery delays negatively impact customer satisfaction and operational efficiency.
This project aims to answer the following questions:

What percentage of orders are delayed?

Which weather conditions cause the most delays?

How does traffic status affect delivery performance?

Does waiting time at warehouses increase delays?

Is there any relationship between inventory levels and delivery delays?

🛠 Tools & Technologies

SQL – Data exploration and analysis

Power BI – Data visualization and dashboard creation

Excel / CSV Dataset – Data source

📊 Dataset Description

The dataset contains logistics delivery information with the following key fields:

order_id – Unique identifier for each order

weather_condition – Weather during delivery

traffic_status – Traffic conditions during shipment

waiting_time – Waiting time before dispatch

inventory_level – Inventory availability

logistics_delay – Delivery status (0 = On Time, 1 = Delayed)

🔎 Key Analysis Performed
1️⃣ Delay Rate Analysis

Calculated the overall percentage of delayed orders to measure delivery performance.

2️⃣ Weather Impact on Delays

Analyzed how different weather conditions affect the probability of delivery delays.

3️⃣ Traffic vs Delivery Performance

Studied how traffic congestion contributes to delayed shipments.

4️⃣ Waiting Time Analysis

Evaluated whether higher waiting time at dispatch locations increases delays.

5️⃣ Inventory vs Delay Relationship

Analyzed whether low inventory levels affect shipment timing.

📈 Key Insights

Bad weather conditions significantly increase delivery delays.

High traffic areas show a higher percentage of delayed orders.

Increased warehouse waiting time leads to higher delay probability.

Inventory shortages can slow down order processing and shipment times.

💡 Business Recommendations

Based on the analysis, the following improvements can help reduce delays:

Optimize delivery routes during heavy traffic periods.

Implement weather-based delivery planning to avoid disruptions.

Reduce warehouse waiting time through better dispatch scheduling.

Improve inventory management to ensure product availability before dispatch.

🚀 Project Outcome

This analysis provides actionable insights that can help logistics teams identify delay patterns, optimize operations, and improve delivery efficiency.
